What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing programs to manipulate the content, structure, and style of the page dynamically. Through the DOM, developers can access and modify HTML or XML elements using JavaScript.

How do you select an element from the DOM?

getElementById(): Selects an element by its ID.
getElementById(): getElementById(): Selects all elements with a specific class name (returns a collection).
getElementsByTagName(): Selects all elements with a specific tag name (returns a collection).
querySelector(): Selects the first element that matches a CSS selector.
querySelectorAll(): Selects all elements that match a CSS selector (returns a NodeList).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element, allowing it to handle events from its child elements using event bubbling. It's useful for improving performance, simplifying code, and managing dynamic content, as you don't need to add separate listeners to each child.
